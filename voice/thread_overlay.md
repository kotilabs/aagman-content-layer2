# Thread Overlay

X threads are compressed arguments. Each post must stand alone if quoted, but together they build a single thesis.

## Universal rules

- Voice: Financial Times leader column meets Matt Levine — dry, confident, structurally argued, lightly literary.
- Brevity is power. Every word earns its place.
- One idea per post.
- Lead with the concrete, follow with the abstract.
- Every post should have one signal sentence — short, declarative, screenshot-quotable.
- Numbers carry the argument. Specific, sourced inline.
- Use em dashes for pivots. Use italics sparingly.
- Cut transitional phrases.
- No adjectives where a number will do.
- Vary post weight: dense posts earn a sparse one-liner next to them.

## Structure

- **Post 1 — Hook:** Name the shift, not the topic. One sharp sentence. No rhetorical questions. No "imagine if."
- **Posts 2 to N-1 — Body:** Each post advances the argument by one beat.
- **Pivot post (mid-thread):** Name the thesis explicitly, often as "This isn't X. It's Y."
- **Post N — Open end:** End with unresolved cognitive tension. No CTA. No "follow for more."

## Formatting

- Number posts as 1/, 2/, 3/ etc.
- Short sentences.
- Generous white space.
- Line breaks for emphasis.
- 0–1 hashtags max, or none.
- No links in the first post.
- Each post under 250 characters where possible.

## Forbidden

- Stock tips, price targets, return promises
- "AI will beat the market" framing
- Urgency bait
- Rhetorical questions mid-thread
- "Imagine if…" / "The future of X"
- Exclamation marks
- Emoji
- Hype openers
- Engagement-bait ("Agree?", "Reply with your take")

## Length

8–15 posts. Let the argument decide.
